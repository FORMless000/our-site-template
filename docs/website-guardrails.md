# Website Guardrails

This site should keep a distinct identity and avoid drifting back toward borrowed template structure.

## Visual direction

- Favor a studio/showcase tone over SaaS-template language.
- Use original copy and avoid inherited marketing phrases.
- Keep the homepage anchored around one featured interactive surface with supporting narrative.
- Prefer a compact custom design system over importing large external UI scaffolds.

## Provenance rules

- Do not add logos, screenshots, icons, or background images copied from unlicensed sources.
- If an asset source is unclear, replace it with a newly created or clearly licensed asset.
- When rebuilding UI patterns, rewrite the markup and styling rather than porting files wholesale.

## Chat behavior

- The chatbot is a floating site tool.
- The fake local assistant can later be replaced with a real model adapter, but the overlay behavior should remain stable.
- Do not restore a full-page chat demo route unless explicitly requested.
