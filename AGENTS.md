# Agent Entry Rules

Read these files before expanding the site:

- `docs/website-guardrails.md`
- `docs/demo-integration.md`

Core rules:

1. Treat this repo as a clean-room rebuild. Do not import code, assets, or text from unlicensed upstream templates.
2. Prefer small custom components over large copied UI kits.
3. Keep popup chat as a site-wide overlay, not a demo route.
4. Keep homepage visuals embeddable and reusable.
5. Preserve local hosting and static export compatibility.
